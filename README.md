
#### 1) What is the difference between `null` and `undefined`?
= null মানে হচ্ছে একদম খালি কিছুই নাই ইচ্ছা করে খালি রাখা হয়েছে আর undefined মানে হচ্ছে জায়গা আছে কিন্তু এখনো কোনো ভ্যালু অ্যাসাইন করা হয়নি। 

#### 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?
= forEach() লুপ চালয় কিন্তু কিছু ফেরত দেয় না আর কিছু করেও না। map() লুপ চালানো পর একটা array দেয়

#### 3) What is the difference between `==` and `===`?
= `===` টাইপ চেক করে। `==` টাইপ চেক করে।

#### 4) What is the significance of `async`/`await` in fetching API data?
= `async` ফাংশনের আগে বসে এবং একটা প্রমিস দেয় `await` ফাংশনের ভেতরে বাসে এবং অপেক্ষা করায়

#### 5) Explain the concept of Scope in JavaScript (Global, Function, Block).
= Global Scope মানে ভেরিয়েবল সবাই ব্যবহার করতে পারবে Block Scope মানে ভেরিয়েবল শুধু এই Block এর মধ্যেই সীমাবদ্ধ থাকবে বাইরে থেকে কেউ ব্যবহার করতে পারবে না।
